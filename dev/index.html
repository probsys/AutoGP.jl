AutoGP.jl

A Julia package for learning the covariance structure of Gaussian process time series models.

Installation

First, obtain Julia 1.8 or later, available here.

The AutoGP package can be installed with the Julia package manager. From the Julia REPL, type ] to enter the Pkg REPL mode and then run:

pkg> add AutoGP
